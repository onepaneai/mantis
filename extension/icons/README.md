# Extension Icons

Please add the following icon files to this directory:

- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

## Creating Icons

You can create these using any image editor. Suggested design:

- Use a shield 🛡️ or lock 🔒 symbol
- Color scheme: Blue (#3498db) and white
- Simple, recognizable design
- Transparent background

## Online Tools

- https://www.favicon-generator.org/
- https://realfavicongenerator.net/
- https://www.iconarchive.com/

## Quick Solution

For testing, you can use any 16x16, 48x48, and 128x128 PNG images temporarily.
